# Mentorship System Documentation

## Overview

The Mentorship System connects founders with experienced advisors and experts for guidance, support, and knowledge sharing.

## üéØ Features

### For Founders

#### Browse Available Experts
1. Go to Mentorship (`/(tabs)/mentorship`)
2. View expert cards:
   - Name and photo
   - Specialization
   - Experience
   - Rating
   - Availability
   - Session price (if applicable)

#### Filter Experts
Filter by:
- **Industry**: Technology, Healthcare, etc.
- **Specialization**: Strategy, Product, Marketing, etc.
- **Availability**: Available now, This week, etc.
- **Rating**: 4+ stars, 5 stars only
- **Price**: Free, Paid, All

#### Send Mentorship Request
1. Click on expert profile
2. Click "Request Mentorship"
3. Fill request form:
   - **Topic**: What you need help with
   - **Preferred Date**: Select date
   - **Preferred Time**: Select time slot
   - **Duration**: 30min, 1hr, 2hr
   - **Message**: Brief description
4. Submit request
5. Wait for expert response

#### View Request Status
- **Pending**: Awaiting expert response
- **Accepted**: Session confirmed
- **Declined**: Expert declined
- **Completed**: Session finished
- **Cancelled**: Request cancelled

#### Manage Sessions
- **Upcoming**: Scheduled sessions
  - View details
  - Reschedule if needed
  - Cancel if necessary
  - Add to calendar
- **Past**: Completed sessions
  - View session notes
  - Access recordings (if available)
  - Leave review

#### Receive Reviews
- After session, expert can rate you
- View ratings in Reviews tab
- Respond to reviews
- Build reputation

### For Experts

#### View Mentorship Requests
1. Go to Expert Sessions (`/(tabs)/expert-sessions`)
2. View "Pending Requests" tab
3. See all requests:
   - Founder name
   - Startup name
   - Request topic
   - Preferred date/time
   - Duration
   - Message

#### Accept/Decline Requests
1. Review request details
2. Click "Accept" or "Decline"
3. If accepting:
   - Confirm date/time
   - Add notes
   - Send confirmation
4. If declining:
   - Optionally provide reason
   - Suggest alternative time

#### Schedule Meetings
- View accepted requests
- Confirm meeting details
- Reschedule if needed
- Add to calendar
- Send reminders

#### Session Calendar
- View all sessions
- See availability
- Block unavailable times
- Manage time slots

#### Leave Reviews
1. After session completion
2. Rate founder (1-5 stars)
3. Write review (optional)
4. Submit feedback
5. Help build founder reputation

#### Session Analytics
- Total sessions hosted
- Hours mentored
- Founders helped
- Average rating
- Response rate

3. **Respond**: Accept or decline

5. **Host**: Conduct session
6. **Review**: Rate founder
