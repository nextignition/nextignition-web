# Admin Dashboard Guide

## Overview

The Admin Dashboard provides platform administrators with comprehensive tools to manage users, content, analytics, and system configuration.

## üéØ Key Features

### A. User Management

#### View All Users
1. Go to Admin Dashboard
2. Navigate to Users section
3. View user list with filters:
   - Role (Founder, Investor, Expert, Admin)
   - Status (Active, Inactive, Suspended)
   - Subscription tier
   - Registration date
   - Last active

#### User Details
- Full profile information
- Account status
- Activity history
- Subscription details
- Security settings
- Support tickets

#### Edit User Profiles
- Update user information
- Change roles (with restrictions)
- Modify subscription
- Suspend/activate accounts
- Reset passwords

#### User Actions
- **Activate**: Activate suspended accounts
- **Suspend**: Temporarily suspend accounts
- **Delete**: Permanently delete accounts
- **Reset Password**: Force password reset
- **Send Email**: Send system emails
