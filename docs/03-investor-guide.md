# Investor Dashboard Guide

## Overview

The Investor Dashboard enables investors to discover startups, review pitch materials, connect with founders, and manage their investment portfolio.

## üéØ Key Features

### A. Profile & Portfolio

#### View/Edit Investor Profile
1. Go to Investor Profile (`/(tabs)/investor-profile`)
2. Edit profile information:
   - **Full Name**: Your name
   - **Fund/Company**: Investment firm or company name
   - **Title**: Your role/title
   - **Bio**: Professional background
   - **Investment Focus**: Areas of interest
   - **Portfolio**: List portfolio companies
   - **Social Links**: LinkedIn, Twitter, Website
   - **Location**: Geographic focus

#### Fund/Company Info
- Company name
- Fund size
- Investment stage focus
- Geographic focus
- Industry focus
- Typical check size

#### Investment Focus
- Select investment stages:
  - Pre-seed
  - Seed
  - Series A
  - Series B+
- Select industries:
  - Technology
  - Healthcare
  - FinTech
  - E-commerce
  - SaaS
  - Others

#### Portfolio Companies
- Add portfolio companies
- Display company logos
- Link to company websites
- Show investment details (optional)

### B. Startup Discovery

#### Browse Startups
1. Go to Startup Discovery (`/(tabs)/startup-discovery`)
2. View startup cards in grid/list view
3. See key information:
   - Startup name
   - Industry
   - Stage
   - Location
   - Funding required
   - Founder name

#### Search Startups
1. Use search bar at top
2. Search by:
   - Startup name
   - Industry
   - Founder name
   - Keywords

#### Filter Startups
Filter by:
- **Stage**: Idea, MVP, Seed, Series A, etc.
- **Location**: City, Country, Region
- **Industry**: Technology, Healthcare, etc.
- **Funding Required**: Amount range
- **Date Added**: Recent, Last week, Last month

#### View Detailed Founder Profiles
1. Click on startup card
2. View full founder profile:
   - Background and experience
   - Previous companies
   - Education
   - Social links
   - Contact information

#### Pitch Deck Preview
1. Click "View Pitch Deck" on startup card
2. Embedded PDF viewer opens
3. Navigate through slides
4. Download option (if permitted)
5. Full-screen view available

#### Pitch Video Preview
1. Click "Watch Video" on startup card
2. Embedded video player opens
3. Play/pause controls
4. Full-screen option
5. 2-minute pitch video

#### Flag/Bookmark Startups
1. Click bookmark icon on startup card
2. Startup saved to "Saved" list
3. Access saved startups from menu
4. Organize with tags/notes

#### Access Pitch History
1. Go to Startup Discovery > History
2. View all startups you've viewed
3. See engagement stats:
   - View count
   - Time spent
   - Actions taken

#### Engagement Stats
- Startups viewed
- Pitch decks opened
- Videos watched
- Connections made
- Meetings scheduled

### C. Networking & Connections

#### Send Connection Requests
1. View startup/founder profile
2. Click "Connect" button
3. Add optional message
4. Send connection request
5. Wait for founder acceptance

#### Manage Connections
1. Go to Network tab
2. View connections:
   - **Accepted**: Mutual connections
   - **Pending**: Awaiting response
   - **Sent**: Your pending requests

#### Past Chat History
1. Go to Chat tab
2. View all conversations
3. Search chat history
4. Filter by:
   - Date
   - Founder name
   - Startup name

#### Engagement Analytics
- Total connections
- Response rate
- Average response time
- Active conversations
- Meeting conversion rate

### D. Events

#### Join Webinars/Events
1. Go to Webinars (`/(tabs)/webinars`)
2. Browse upcoming events
3. Filter by topic/date
4. Click "Register"
5. Receive calendar invite

#### Access Recordings
1. Go to Webinars > Recordings
2. View past webinar recordings
3. Access based on subscription:
   - **Free**: Limited access
   - **Pro**: Full access
   - **Elite**: Full access + exclusive

#### Host Sessions
1. Apply to host investor sessions
2. Schedule office hours
3. Manage attendee list
4. Host group sessions with founders

### E. Platform News & Profile Summarizer

#### Investment Opportunities News
1. Go to Feed tab
2. View filtered news:
   - New startup launches
   - Funding rounds
   - Investment opportunities
   - Market trends

#### Advanced Summarizer Agent
1. Go to Documents > AI Summarizer
2. Enter startup/founder name
3. AI generates summary:
   - Profile overview
   - Key metrics
   - Investment highlights
   - Risk factors
   - Recommendation

#### Quick Review
- One-click profile summary
- Key metrics extraction
- Investment thesis generation
- Risk assessment

### F. Subscription Management

#### View Current Plan
1. Go to Subscription (`/(tabs)/subscription`)
2. View plan details:
   - **Free**: Basic discovery
   - **Pro**: Advanced features
   - **Elite**: Premium access
3. See renewal date

#### Payment History
1. Go to Subscription > Payments
2. View all transactions
3. Download invoices
4. View payment methods

#### Feature Access
- **Free**: Basic startup discovery
- **Pro**: Advanced filters, AI summarizer
- **Elite**: Priority access, exclusive deals

- Time spent per startup

#### Investment History
- Connections made
