## üîÑ Subscription Management

### View Current Plan
1. Go to Subscription (`/(tabs)/subscription`)
2. View plan details:
   - Plan name
   - Features included
   - Renewal date
   - Billing cycle

### Upgrade Plan
1. Go to Subscription
2. Click "Upgrade"
3. Select new plan
4. Review features
5. Enter payment details
6. Confirm upgrade
7. Plan activated immediately

### Downgrade Plan
1. Go to Subscription
2. Click "Change Plan"
3. Select lower plan
4. Confirm downgrade
5. Plan changes at next billing cycle

### Cancel Subscription
1. Go to Subscription
2. Click "Cancel Subscription"
3. Confirm cancellation
4. Access continues until period ends
5. No refunds for current period

## üí∞ Payment Management

### Payment Methods
- Credit/Debit cards
- PayPal (future)
- Bank transfer (annual)
- Digital wallets (future)

### Add Payment Method
1. Go to Subscription > Payment Methods
2. Click "Add Payment Method"
3. Enter card details
4. Save securely
5. Set as default (optional)

### Update Payment Method
1. Go to Subscription > Payment Methods
2. Select payment method
3. Click "Edit"
4. Update details
5. Save changes

### Remove Payment Method
1. Go to Subscription > Payment Methods
2. Select payment method
3. Click "Remove"
4. Confirm removal
