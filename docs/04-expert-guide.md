# Expert/Advisor Dashboard Guide

## Overview

The Expert Dashboard enables advisors and mentors to manage their profile, receive mentorship requests, host sessions, and build their reputation on the platform.

### A. Profile & Experience Management

#### View/Edit Expert Profile
1. Go to Expert Profile (`/(tabs)/expert-profile`)
2. Edit profile information:
   - **Full Name**: Your name
   - **Title**: Current role/title
   - **Experience**: Years of experience
   - **Specialization**: Areas of expertise
   - **Portfolio**: Past companies/projects
   - **Mentorship Availability**: Hours per week
   - **Skills**: Technical/business skills
   - **Industries**: Industries you work in
   - **Social Links**: LinkedIn, Twitter, Website
   - **Bio**: Professional background

#### Experience Details
- Years of experience
- Previous roles
- Companies worked for
- Notable achievements
- Certifications
- Education

#### Specialization
- Select areas:
  - Business Strategy
  - Product Development
  - Marketing & Growth
  - Fundraising
  - Technical Architecture
  - Operations
  - Sales
  - Others

#### Portfolio
- Add past companies
- Add notable projects
- Include success metrics

#### Mentorship Availability
- Hours per week available
- Preferred time slots
- Timezone
- Session types (1-on-1, group)

#### Skills & Industries
- Add relevant skills
- Select industries you work in
- Set expertise level for each

### B. Mentorship & Session Management

#### View Founder Requests
1. Go to Expert Sessions (`/(tabs)/expert-sessions`)
2. View "Pending Requests" tab
3. See all mentorship requests:
   - Founder name
   - Startup name
   - Request topic
   - Preferred date/time
   - Duration
   - Message from founder

#### Accept/Decline Sessions
1. Review request details
2. Click "Accept" or "Decline"
3. If accepting:
   - Confirm date/time
   - Add notes
   - Send confirmation message
4. If declining:
   - Optionally provide reason
   - Suggest alternative time

#### Schedule Meetings
1. Go to Upcoming Sessions
2. View accepted sessions
3. Reschedule if needed
4. Add to calendar
5. Send reminders

#### Session Calendar
- View all sessions in calendar
- Filter by status
- See availability
- Block unavailable times

#### Session History
1. Go to Past Sessions
2. View completed sessions:
   - Date/time
   - Founder name
   - Topic discussed
   - Duration
   - Notes (if added)

#### Leave Ratings/Reviews
1. After session completion
2. Rate founder (1-5 stars)
3. Write review (optional)
4. Submit feedback
5. Help build founder reputation

### C. Webinars & Visibility

#### Host Webinars
1. Go to Host Webinar (`/(tabs)/host-webinar`)
2. Create new webinar:
   - **Title**: Webinar title
   - **Description**: What it covers
   - **Date & Time**: Schedule
   - **Duration**: Length (30min, 1hr, etc.)
   - **Max Attendees**: Limit (optional)
   - **Price**: Free or paid
   - **Topics**: What you'll cover
3. Click "Create Webinar"
4. Share webinar link
5. Manage registrations

#### Join Webinars
1. Go to Webinars tab
2. Browse available webinars
3. Register for interesting sessions
4. Attend live or watch recordings

#### Webinar Scheduling
- Schedule recurring sessions
- Set up office hours
- Create webinar series
- Manage multiple webinars

#### Session Tracker
- View all webinars:
  - Upcoming
  - Past
  - Cancelled
- See registration numbers
- Track attendance

#### Access Session Library
1. Go to Webinars > Recordings
2. View past webinar recordings
3. Access your hosted sessions
4. Download recordings (if permitted)

#### Direct Message Users
1. Go to Chat tab

#### Notifications & Unread Badges
