# Founder/Co-founder Dashboard Guide

## Overview

The Founder Dashboard is designed for startup founders and co-founders to manage their startup profile, seek funding, connect with investors, and access mentorship.

## ðŸŽ¯ Key Features

### A. Profile & Startup Management

#### View/Edit Profile
1. Navigate to Profile tab
2. Click "Edit Profile" button
3. Update information:
   - Full name
   - Email (cannot change)
   - Bio/description
   - Location
   - Social links (LinkedIn, Twitter, Website)
   - Profile picture
4. Click "Save Changes"

#### Startup Details Panel
1. Go to Startup Profile (`/(tabs)/startup-profile`)
2. Manage startup information:
   - **Startup Name**: Your company name
   - **Description**: What your startup does
   - **Stage**: Idea, MVP, Seed, Series A, etc.
   - **Website**: Company website URL
   - **Industry**: Select from dropdown
   - **Pitch Deck**: Upload PDF (max 10MB)
   - **Pitch Video**: Record/upload 2-minute video
   - **Visibility**: Public/Private toggle

#### Profile Completeness Tracker
- Visual progress bar showing profile completion
- Complete all sections for 100% completion
- Higher completion = better visibility to investors

#### Profile Share/Export
- **Share Link**: Generate shareable profile link
- **Export PDF**: Download profile as PDF
- **Social Share**: Share to LinkedIn, Twitter

### B. Funding Portal

#### Submit Pitch Deck
1. Go to Funding Portal (`/(tabs)/funding`)
2. Click "Upload Pitch Deck"
3. Select PDF file (max 10MB)
4. Set visibility (Public/Private)
5. Click "Upload"
6. Deck appears in your portfolio

#### Submit Pitch Video
1. Go to Funding Portal > Pitch Video
2. Click "Record Video" (2-minute limit)
3. Record or upload video
4. Preview before submitting
5. Click "Submit"
6. Video appears in your portfolio

#### Status Tracker
1. Go to Funding Status (`/(tabs)/funding-status`)
2. View all funding requests:
   - **Pending**: Awaiting investor review
   - **Viewed**: Investors have viewed your pitch
   - **Interested**: Investors expressed interest
   - **Meeting Scheduled**: Meetings arranged
   - **Funded**: Successfully funded

#### Connect with Investors
1. View interested investors list
2. Click investor profile
3. Send connection request
4. Start chat conversation
5. Schedule meetings

#### View Responses/Messages
- Check Messages tab for investor responses
- Reply to investor inquiries
- Manage conversation history

#### Meeting Scheduler
1. Go to Funding Status > Meetings
2. View scheduled meetings
3. Accept/decline meeting requests
4. Add to calendar
5. Join video calls

#### Funding History & Analytics
- **Views**: How many investors viewed your pitch
- **Connections**: Investor connection requests
- **Meetings**: Scheduled meetings
- **Success Rate**: Funding success metrics

### C. Mentorship & Networking

#### View Available Experts
1. Go to Mentorship (`/(tabs)/mentorship`)
2. Browse available experts/advisors
3. Filter by:
   - Industry
   - Specialization
   - Availability
   - Rating

#### Send Mentorship Request
1. Click on expert profile
2. Click "Request Mentorship"
3. Fill request form:
   - Topic/area of help needed
   - Preferred date/time
   - Duration (30min, 1hr, etc.)
   - Brief description
4. Submit request
5. Wait for expert response

#### View Past/Current Sessions
- **Upcoming**: Scheduled sessions
- **Past**: Completed sessions
- **Cancelled**: Cancelled sessions
- View session details and notes

#### Receive Ratings/Reviews
- After mentorship session, expert can rate you
- View ratings in Reviews tab
- Respond to reviews
- Build your reputation

#### Network Feed
- Latest connections
- Session invites
- Onboarding/funding success stories
- Platform announcements

### D. Events & Webinars

#### See Upcoming Sessions
1. Go to Webinars (`/(tabs)/webinars`)
2. View "Upcoming" tab
3. See all scheduled webinars
4. Filter by:
   - Date
   - Topic
   - Host
   - Price (Free/Paid)

#### Register for Events
1. Click on webinar
2. View details (title, description, date, time)
3. Click "Register"
4. Confirm registration
5. Receive calendar invite

#### Access Recordings
1. Go to Webinars > Recordings tab
2. View past webinar recordings
3. Access based on subscription tier:
   - **Free**: Limited access
   - **Pro**: Full access
   - **Elite**: Full access + exclusive content

#### Host Events (If Eligible)
- Apply to host webinars
- Schedule sessions
- Manage attendees
- Access analytics

### E. Community Chat

#### Join Founders' Group Chat
1. Go to Chat tab
2. View available channels
3. Join "Founders" channel
4. Start participating in discussions

#### Direct Message Users
1. Go to Chat tab
2. Click "New Message"
3. Search for user
4. Start conversation
5. Send messages, files, links

#### Notifications & Unread Badges
- Unread message count in tab badge
- Push notifications for new messages
- Email notifications (optional)

#### Search Chat History
1. Open conversation
2. Click search icon
3. Enter search term
4. View matching messages
5. Jump to message

### F. Subscription & Payments

#### View Current Plan
1. Go to Subscription (`/(tabs)/subscription`)
2. View current plan:
   - **Free**: Basic features
   - **Pro**: Advanced features
   - **Elite**: Premium features
3. See renewal date
4. View plan benefits

#### Payment History
1. Go to Subscription > Payment History
2. View all transactions
3. Download invoices
4. View payment methods

#### Upgrade/Downgrade
1. Go to Subscription
2. Click "Upgrade" or "Change Plan"
3. Select new plan
4. Enter payment details
5. Confirm change
6. Plan updated immediately

### G. Notifications & Activity Feed

#### Notification Types
- **Funding**: Investor interest, meeting requests
- **Sessions**: Mentorship invites, session reminders
- **Reviews**: New ratings received
- **System**: Platform updates, verification

#### Activity Feed
1. Go to Feed (`/(tabs)/feed`)
2. View automated activity posts:
   - New connections
   - Funding milestones
   - Session completions
   - Platform updates

#### Manage Notifications
1. Go to Notifications (`/(tabs)/notifications`)
2. View all notifications
3. Mark as read
4. Dismiss notifications
5. Configure preferences in Settings
