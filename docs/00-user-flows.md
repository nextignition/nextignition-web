# Complete User Flows

## Overview

This document outlines all user flows across the NextIgnition platform, organized by user role and feature.

### New User Registration
1. **Sign Up**
   - Navigate to `/register`
   - Enter email and password
   - Accept terms
   - Submit registration

2. **Email Verification**
   - Receive verification email
   - Click verification link
   - Account activated

3. **Role Selection**
   - Choose role (Founder, Investor, Expert)
   - Complete role-specific setup

4. **Profile Setup**
   - Add basic information
   - Upload profile picture
   - Set location
   - Complete onboarding

### Login Flow
1. Navigate to `/login`
2. Enter email and password
3. Click "Sign In"
4. Redirected to role-specific dashboard

### Password Reset
1. Click "Forgot Password?"
2. Enter email
3. Receive reset link
4. Set new password
5. Login with new password
