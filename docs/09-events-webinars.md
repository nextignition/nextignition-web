# Events & Webinars Documentation

## Overview

The Events & Webinars system enables users to host, join, and access recordings of educational sessions, networking events, and expert talks.

## üéØ Features

### For All Users

#### Browse Webinars
1. Go to Webinars (`/(tabs)/webinars`)
2. View available webinars:
   - **Upcoming**: Future sessions
   - **Recordings**: Past sessions
3. Filter by:
   - Date
   - Topic
   - Host
   - Price (Free/Paid)
   - Industry

#### Register for Events
1. Click on webinar
2. View details:
   - Title and description
   - Date and time
   - Duration
   - Host information
   - Topics covered
   - Price
3. Click "Register"
4. Confirm registration
5. Receive calendar invite

#### Access Recordings
1. Go to Webinars > Recordings
2. Browse past webinars
3. Filter by:
   - Date
   - Topic
   - Host
4. Click to watch
5. Access based on subscription tier

### For Hosts (Experts/Admins)

#### Host Webinar
1. Go to Host Webinar (`/(tabs)/host-webinar`)
2. Create new webinar:
   - **Title**: Webinar title
   - **Description**: What it covers
   - **Date**: Select date
   - **Time**: Select time
   - **Duration**: 30min, 1hr, 2hr
   - **Max Attendees**: Limit (optional)
   - **Price**: Free or set price
   - **Topics**: What you'll cover
3. Click "Create Webinar"
4. Webinar appears in listings

#### Manage Webinars
- View all webinars:
  - Upcoming
  - Past
  - Cancelled
- Edit webinar details
- Cancel webinars
- Manage registrations

#### Session Management
- View registered attendees
- Send reminders
- Manage waiting list
- Handle cancellations
- Track attendance
