## Admin Panel Features

### Dashboard
- View platform overview metrics
- Analytics widgets with real-time data
- Chart visualizations (placeholders)

### User Management
- Approve/reject pending users
- Search users by name or email
- Filter by status (All, Pending, Approved, Rejected)
- View user details and activity

### Reports & Flags
- Manage user reports (spam, harassment, fake profiles)
- Priority-based filtering (High, Medium, Low)
- Resolve or dismiss reports
- Search and filter functionality

### Analytics
- Comprehensive analytics dashboard
- User growth metrics
- Revenue tracking
- Engagement statistics
- Trend indicators

## Navigation

After logging in with admin credentials, you'll be automatically redirected to:
- `/(admin)/dashboard` - Main admin dashboard

You can navigate between admin screens using the routes:
- `/(admin)/dashboard` - Dashboard
- `/(admin)/users` - User Management
- `/(admin)/reports` - Reports & Flags
- `/(admin)/analytics` - Analytics
